5、在 Cursor 里怎么连？（操作步骤）

到这一步，Cursor 本质上只是跑一条 ssh ubuntu110 ...，跟你命令行一样，所以连接会非常稳。

方式 A：用 Host 名连接（推荐）

  1. 打开 Cursor。

  2. 找到 Remote / SSH 面板（跟 VSCode 的 Remote-SSH 很像）。

  3. 在 Host 列表里，你应该能看到 ubuntu110、ubuntu100（它是从 ~/.ssh/config 里读的）。

  4. 直接点 ubuntu110 → Cursor 内部会执行：

    因为 ubuntu110 在 config 里已经指定了 IdentityFile，
    ssh 会用这把 key，不会再走密码/askpass，连接就能成功。

方式 B：手动填信息（如果它不给你选 Host）

如果 Cursor 弹出的是一个表单，让你填 Host/User/Port/Auth，那就这样填：

  • Host：192.168.135.110
  • User：lixiaodao
  • Port：22
  • Auth / Authentication：
    • 选择「使用私钥 / Key」
    • 文件路径选：C:\Users\ZhuanZ（无密码）\.ssh\id_ed25519

想连另一台就改成 192.168.135.100 + lixiaodao1234。
