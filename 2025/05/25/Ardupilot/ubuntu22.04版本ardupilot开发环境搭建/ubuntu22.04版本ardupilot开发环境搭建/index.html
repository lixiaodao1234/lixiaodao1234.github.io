6.配置gitee管理Ardupilot项目

初次运行 Git 前的配置

既然已经在系统上安装了 Git，你会想要做几件事来定制你的 Git 环境。 每台计算机上只需要配置一次，程序升级时会保留配置信息。 你可以在任何时候再次通过运行命令来修改它们。

Git 自带一个 git config 的工具来帮助设置控制 Git 外观和行为的配置变量。 这些变量存储在三个不同的位置：

  1. /etc/gitconfig 文件: 包含系统上每一个用户及他们仓库的通用配置。 如果在执行 git config 时带上 --system 选项，那么它就会读写该文件中的配置变量。 （由于它是系统配置文件，因此你需要管理员或 超级用户权限来修改它。）
  2. ~/.gitconfig 或 ~/.config/git/config 文件：只针对当前用户。 你可以传递 --global 选项让 Git 读写此文件，这会对你系统上 所有 的仓库生效。
  3. 当前使用仓库的 Git 目录中的 config 文件（即 .git/config）：针对该仓库。 你可以传递 --local 选 项让 Git 强制读写此文件，虽然默认情况下用的就是它。（当然，你需要进入某个 Git 仓库中才能让该选 项生效。）

每一个级别会覆盖上一级别的配置，所以 .git/config 的配置变量会覆盖 /etc/gitconfig 中的配置变量。

在 Windows 系统中，Git 会查找 $HOME 目录下（一般情况下是 C:\Users\$USER ）的 .gitconfig 文件。 Git 同样也会寻找 /etc/gitconfig 文件，但只限于 MSys 的根目录下，即安装 Git 时所选的目标位置。 如果 你在 Windows 上使用 Git 2.x 以后的版本，那么还有一个系统级的配置文件，Windows XP 上在 C:\Documents and Settings\All Users\Application Data\Git\config ，Windows Vista 及更新 的版本在 C:\ProgramData\Git\config 。此文件只能以管理员权限通过 git config -f 来修 改。

6.1 Gitee生成SSH密钥对

  1. 打开终端（Windows 用户可使用 Git Bash 或 PowerShell）：
    • Windows：安装 Git 后，右键菜单选择 Git Bash Here。
    • Mac/Linux：直接打开终端。
  2. 输入生成密钥的命令：
  • -t rsa：指定密钥类型为 RSA。
  • -C "注释"：替换为你的邮箱（仅作标识，不影响功能）。
  1. 设置保存路径和密码：
    • 提示 Enter file in which to save the key 时，按回车使用默认路径（通常为 ~/.ssh/id_rsa）。
    • 输入密钥密码（可选，直接回车可跳过，但建议设置密码提高安全性）。
  2. 生成成功：
    • 密钥对会保存在 ~/.ssh/ 目录下：
      • 私钥：id_rsa（切勿泄露！）
      • 公钥：id_rsa.pub

7.1 参数指令

-v (Vehicle): ArduCopter (多旋翼), ArduPlane (固定翼), Rover (车), Sub (潜艇)。

-f (Frame): 机型框架，如 + (十字), quad (X字), hexa (六轴), gazebo-iris (连接 Gazebo)。

-L (Location): 启动位置（在 Tools/autotest/locations.txt 中定义），如 Xian, GWTC (广州)。

--console: 打开 MAVProxy 控制台（输入命令的地方）。

--map: 打开调试地图。

-w: 擦除所有参数并恢复默认值（Wipe），当你参数调乱了导致连不上时非常有用。

7.2 网络通信参数 (--out)

前缀 完整示例 逻辑含义 (MAVProxy 的角色) 适用场景
无前缀 --out 10.8.0.5:14550 UDP 客户端：主动把数据推向这个 IP 连同一个局域网的地面站 (MP/QGC)
udpout: --out udpout:10.8.255.2:14550 UDP 客户端：同上，显式声明 VPN 环境最推荐，推给远程 MAVROS
udpin: --out udpin:0.0.0.0:14550 UDP 服务端：在本地开个口子等数据 接收外部发来的特殊 MAVLink 数据
tcp: --out tcp:10.8.0.5:5760 TCP 客户端：主动去连别人的 TCP 端口 极少用，除非对方是 TCP 服务器
tcpin: --out tcpin:0.0.0.0:5762 TCP 服务端：在本地开个 TCP 端口等连 MAVROS 连仿真 的常用方式

7.3 常用 MAVProxy 控制台命令 (启动后输入)

仿真运行起来后，在那个黑色的 Console 窗口里可以输入：

  • 解锁/上锁： arm throttle / disarm
  • 切换模式： mode GUIDED / mode LOITER / mode RTL
  • 起飞（Guided模式下）： takeoff 10 (起飞到10米)
  • 查看参数： param show [参数名] (如 param show BATT_CAPACITY)
  • 修改参数： param set [参数名] [值]
  • 加载参数文件： param load /path/to/your.parm
  • 查看连接状态： status (可以看到链路质量和丢包率)
